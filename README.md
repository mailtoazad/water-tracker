# 💧 Water Tracker

A mobile-friendly water intake tracker built with React + Vite.

## Features
- Track daily water intake
- Adjustable daily goal
- 7-day history view
- Data saved to localStorage (persists across refreshes)

## Live App
Your app will be live at: `https://YOUR_USERNAME.github.io/water-tracker/`

## Local Development
```bash
npm install
npm run dev
```
